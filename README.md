# Bio-Alchemie – Klasse 8

Ein browserbasiertes Lernspiel für Biologie in Klasse 8.

Schülerinnen und Schüler kombinieren biologische Grundbegriffe. Wenn zwei Begriffe fachlich zusammenpassen, entsteht ein neuer Begriff mit kurzer Erklärung.

## Start

Die App läuft direkt im Browser über `index.html`.

## Dateien

- `index.html` – Grundstruktur
- `style.css` – Gestaltung
- `script.js` – Spiellogik und Begriffe

## Themenbereiche

- Startbegriffe
- Zelle & Erbinformation
- Körper & Stoffwechsel
- Ökologie & Pflanzen
- Immunsystem
- Evolution

## Funktionen

- Begriffe kombinieren
- neue Fachbegriffe freischalten
- kurze Erklärsätze
- Fortschrittsspeicherung mit `localStorage`
- Tipp-Funktion
- Abzeichen
- geschützter Neustart

## Hinweis

Die Kombinationen sind spielerisch vereinfacht. Die Erklärtexte stellen den biologischen Zusammenhang her.
